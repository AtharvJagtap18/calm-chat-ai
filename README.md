CalmChat AI – Mental Wellness Support Chatbot

CalmChat AI is a web-based conversational chatbot designed to provide calming and supportive interactions for users experiencing stress or anxiety. The application demonstrates how conversational AI can be integrated into a lightweight web application using Python and Flask.

The system allows users to log in and interact with a chatbot through a simple chat interface. The chatbot processes user messages and generates supportive responses using rule-based natural language processing techniques.

The project focuses on combining AI-based conversational logic with a web application interface, creating a practical example of how chatbot systems can be used in mental wellness support applications.

Features

Web-based chatbot interface

Real-time chat interaction

User login and authentication

Simple NLP-based chatbot responses
