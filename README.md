# CalmChat AI – Mental Wellness Support Chatbot

CalmChat AI is a web-based chatbot designed to provide calming and supportive conversations for users experiencing stress or anxiety.

The application demonstrates how conversational AI can be integrated into a lightweight web application using **Python and Flask**. Users can log in and interact with a chatbot through a simple chat interface where the chatbot generates supportive responses using rule-based natural language processing techniques.

---

## Features

- Web-based chatbot interface  
- Real-time chat interaction  
- User login and authentication system  
- NLP-based chatbot responses  
- Relaxing UI with background visuals  
- Background music for a calming experience  
- Lightweight Flask backend  

---
